function findUnit input String unitToFind "display unit to find in unitConversions"; input UnitConversions.RealTypeRecord[:] unitConversions = UnitConversions.RealTypeDef "Unit conversions"; output Integer typeDef "index in unitConversions"; algorithm typeDef:=0; for i in 1:size(unitConversions,1) loop if unitConversions[i].DisplayUnit == unitToFind then typeDef:=i; break; end if; end for; if typeDef==0 then Modelica.Utilities.Streams.print("Unit \"" + unitToFind + "\" not defined. Do not hesitate to contact me. marek@matfyz.cz\n"); end if; end findUnit;

function GenerateSubstanceUnits input String nonSIunit "non-SI unit"; input Real scale "1 mol = 'scale' 'nonSIunit'"; output RealTypeRecord[size(Prefix,1)*size(Suffix,1)] units "generated conversions for mili-, micro-, .. amount/concentration/flow"; protected constant Integer np = size(Prefix,1); constant Integer ns = size(Suffix,1); algorithm units:={ RealTypeRecord( Unit="mol"+SuffixSI[integer(1+mod(i,ns))], DisplayUnit=Prefix[integer(1+div(i,ns))] + nonSIunit + Suffix[integer(1+mod(i,ns))], Scale=PrefixScale[integer(1+div(i,ns))]*scale*SuffixScale[integer(1+mod(i,ns))]) for i in 0:(np*ns-1)}; end GenerateSubstanceUnits;
